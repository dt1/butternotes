Technology:

Butternotes is written using Clojure, VueJS, PostgreSQL Foundation CSS, and a smattering of vanillla javascript.

You can find the complete source on github

All sheet music, music xml, and audio is generated on the fly.

Music xml is generated with MusicXML

The sheet music is rednered with OpenSheetMusicDisplay

Audio is created with Wad (Wed Audio DAW)
