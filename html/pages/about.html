Technology:

Butternotes is written using Clojure, VueJS, PostgreSQL, Foundation CSS, and a smattering of vanillla javascript. All of the images are created by me.

You can find the complete source for this webiste on github

All sheet music, music xml, and audio is generated on the fly.

Music xml is generated with MusicXML

The sheet music is rednered with OpenSheetMusicDisplay

Audio is created with Wad (Wed Audio DAW)
